# Tip

只能在网页端下载书籍，不支持转换其自定义的书籍格式。

只能下载能看到的书籍内容 —— 如果没有购买书籍，那么只能下载书籍试看的部分。

书籍有两种阅读模式，对应不同的下载格式：

-   原貌：下载的是以图片拼接而成的 PDF，无法复制内部文字
-   流式：下载的是 epub，但不是所有书籍都具备该阅读模式

---

**账号的安全是最重要的**，所以：

-   没用采取【直接访问接口来获取书籍内容】的方案，虽然该方案的书籍下载速度非常快
-   没用重写 `XHR` 等其它任何 Web API
-   仅通过插件拦截响应来下载书籍

具体来说，使用以下方式尽可能保证账号安全：

-   利用浏览器插件读取网页的请求与响应，从而分析、下载内容，和网页环境隔离
-   为了能加载书籍的内容，需要操作阅读页面来触发加载，可以使用插件的自动点击、滑动功能 —— 为了模拟是真人在阅读：
    -   【原貌阅读模式】中，需要进行滑动翻页，每次滑动的间隔默认是 `3 秒`，滑动的距离较短
    -   【流式阅读模式】中，需要进行点击翻页，每次点击的间隔默认是 `3 秒`
    -   上述自动化操作是 JS 触发，没有模拟真人的鼠标运动规则，可能是一个隐患
-   我不清楚下载的书籍中是否有隐藏的水印，如果想分享给他人，请注意其中风险

网站提供专门的阅读软件，其中可以下载书籍，得到的是自定义的书籍格式，也许能将其转换成通用的书籍格式 —— 很遗憾，能力所限我无法解决。
